# DRIVE ELITE - L'excellence sur 4 roues

Bienvenue sur le projet Next.js 14 pour DRIVE ELITE. Ce site a été conçu pour offrir une expérience ultra premium, inspirée des leaders du marché de la mobilité de luxe (Uber Black, Sixt Premium).

## Démarrage Rapide

1. **Installation des dépendances**
Veillez à avoir suffisamment d'espace disque.

```bash
npm install
```

2. **Lancement du serveur de développement**
```bash
npm run dev
```

3. **Visualisation**
Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur.

## Architecture

- **Next.js 14 App Router** : Routage performant et SEO-friendly
- **Tailwind CSS** : Stylisation utility-first, encapsulant le Design System Noir/Or
- **Framer Motion** & **Lucide React** : Micro-animations et iconographie Premium
- **Data Mockée** : Accédez au dossier `src/data` pour modifier les 20 véhicules (`cars.ts`) ou les 6 avis clients (`testimonials.ts`).

## Notes de l'Architecte

Toutes les demandes ont été intégrées pour maximiser la conversion :
- Formulaire immersif avec Date Pickers
- Section Véhicules filtrable
- Call To Actions "Paris Nuit" omniprésents
- Floating Action Button WhatsApp pour l'accessibilité 24/7.
- Modèle asymétrique 16/10 pour les ratios d'image des véhicules.
